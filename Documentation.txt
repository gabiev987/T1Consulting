Инструкция:

1. Запускаете приложение либо через T1-Consulting.jar, либо через IDE
2. Отправляете GET запрос (либо через браузер, либо postman) по адресу:

http://localhost:8080/chars/aabbb

3. Получаете ответ в виде JSON {"a":2, "b":3}

Вместо "aabbb" можете писать любую строку, ответом будет JSON с подсчитанным кол-вом символов в строке.
